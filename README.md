# Dashboard – Controle de Assinaturas Xbox

## 📊 Descrição do Projeto

Este projeto apresenta um **Dashboard de Controle de Assinaturas Xbox**, desenvolvido com o objetivo de analisar e organizar dados de assinaturas, permitindo uma visualização clara de informações como quantidade de usuários, tipos de planos e análise de crescimento.

O dashboard facilita a interpretação dos dados através de gráficos e indicadores, auxiliando na tomada de decisões baseadas em dados.

## 🎯 Objetivo

Demonstrar habilidades em **organização de dados, análise e criação de dashboards**, utilizando ferramentas de análise e visualização.

## 🛠 Tecnologias Utilizadas

* Microsoft Excel
* Dashboard com gráficos e tabelas dinâmicas
* Análise de dados
* Organização de informações

## 📈 Funcionalidades do Dashboard

* Visualização de assinaturas
* Análise de crescimento
* Indicadores de desempenho
* Organização de dados em gráficos e tabelas

## 📷 Visualização

O projeto inclui prints do dashboard demonstrando a análise dos dados.
